# LeafAuto Web - 智能自动化平台

一个现代化的智能自动化平台，采用半透明悬浮效果设计，提供任务管理、AI助手、系统监控等功能。

## ✨ 特性

### 🎨 现代化界面设计
- **半透明悬浮效果**：营造轻盈层次感
- **响应式布局**：支持各种设备尺寸
- **深色/浅色主题**：可自由切换
- **流畅动画**：悬停效果和过渡动画

### 📊 仪表盘
- **实时统计**：任务执行数量、AI交互次数、系统工作时长
- **数据可视化**：任务执行趋势图、AI交互统计图表
- **系统状态**：实时监控系统运行状态

### ⏰ 定时任务管理
- **全生命周期管理**：添加、删除、编辑任务
- **多种执行类型**：一次性、间隔执行、Cron表达式
- **实时状态监控**：查看任务执行历史和状态
- **参数配置**：灵活的任务参数设置

### 🤖 AI 智能助手
- **豆包风格界面**：模仿现代聊天应用设计
- **连续对话**：支持上下文管理
- **智能回复**：根据关键词提供相关帮助
- **指令执行**：支持系统操作指令

### 👤 个人中心
- **个人信息展示**：头像、名称、使用统计
- **系统设置**：主题切换、通知管理、时区设置
- **卡片式布局**：清晰的功能分区

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Flask 2.0+
- 现代浏览器（支持ES6+）

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd LeafAuto_Web
```

2. **创建虚拟环境**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **启动应用**
```bash
python application.py
```

5. **访问系统**
打开浏览器访问：`http://localhost:5000`
